某公司

需要一个直观的总体评价系统，根据 a 的评价进行汇总，具体实现形式有待商讨和考虑

公司圈内的用户 a

发帖人

可发帖对公司进行评价，可选择实名或匿名

如果选择匿名，a 是可以自由组织语言的，比如a说“我们公司996”，其他人员是看不出来a是谁的。但如果a说“老板昨天让我以后996”这样的情况，我们就不考虑了

公司圈内的用户 b

看贴人

看贴人 b 视为一个群体，可对帖子进行评价，帖子对公司的最终影响会视评价而定。如果a发布的是假消息，自会由b进行处理从而消除影响维护公司名誉。

如果a匿名，b无法从内容以外的手段确认a的身份，以保证减轻a的社交压力。

b 可以评论，可实名或匿名

公司圈外的用户 c

发帖人

公司圈外的用户 d

看帖人

c 的发帖，需要门槛。比如，我面试过，可填写面试经历，但同时需要证明，证明可以被查阅，但不能被任何手段复制下载。

同用户 a 的发帖，c 的匿名可以通过控制内容的方式保护自己的隐私，但一定要有证据

可以查看公司下的所有帖子，并进行评论。d的评论对公司总体评价不做影响

对于 a 的帖子，a 属于公司，对 c 来说自然有一定的真实性，并可根据 b 的总体评价对信息进行自我判断

对于 c 的帖子，可以通过查看证据进行判断。证据的有效性影响 d 自己得出的最终评论

公司圈内的用户 e

公司形象的捍卫者

e 可以对 a 或 c 的帖子进行解释或道歉。e 可以是公司圈内的任何人。解释或道歉可以对公司最终评价做出正面的影响

用户 f

恶意中伤公司者

公司圈内的f：多半是想离职或者已经离职的人了。公司圈内可集体投票踢出f，投票完全保密没有人能知道谁投的是什么

f 过分恶意中伤公司，公司可申请起诉 f ，我们将在法律认定 f 有责的情况下公开 f 的个人信息

公司圈外的 f: 发帖是要进公司圈的。就算混进去也是会被提出来的。作为 c 发帖，也是需要证据的综合判断对公司的影响不大
